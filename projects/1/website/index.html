Static Website on AWS

Demonstration project showing how to host a static website on AWS using S3 and CloudFront, deployed as Infrastructure as Code with Terraform.

🗂️
Amazon S3
Stores the static files. Bucket is private — no direct public access.
🌍
CloudFront
Global CDN that serves the site over HTTPS from edge locations worldwide.
🔐
Origin Access Control
Only CloudFront can read the S3 bucket. Users never access S3 directly.
⚙️
Terraform
All infrastructure defined as code. Deploy everything with a single command.
